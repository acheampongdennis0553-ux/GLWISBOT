    ────────────────

──────────────────────────────────────────────────────────
    Go to: http://localhost:5000

🌐 ACCESSING THE BOT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

From Your Computer (Local)
──────────────────────────
   Browser URL: http://localhost:5000
   OR:          http://127.0.0.1:5000
   Method:      Any web browser


From Another Computer on Same Network
──────────────────────────────────────
   1. Edit app.py, change to:
      app.run(host='0.0.0.0', port=5000)
   
   2. Find your computer's IP:
      Windows: ipconfig (find IPv4 Address)
      Mac/Linux: ifconfig
      Example: 192.168.1.100
   
   3. From other computer go to:
      http://192.168.1.100:5000


Deployed on Cloud/Server
────────────────────────
   Browser URL: https://your-domain.com
   (Requires cloud deployment setup)

   ────────────────────────────────────────
   File: .env (in same directory as app.py)
   Content:

2️⃣  Install Flask (if not done):
   ───────────────────────────
   pip install Flask


3️⃣  Make sure you're in the right directory:
   ──────────────────────────────────────
   cd c:\Users\DENNY\AI\GLWIS


4️⃣  Run the web server:
   ───────────────────
   python run_web.py
   
   OR
   
   python app.py


5️⃣  Open browser to http://localhost:5000
   ──────────────────────────────────────

🔧 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Problem                        Solution
─────────────────────────────────────────────────────────────────────────────
"No module named 'flask'"      pip install Flask

"Connection refused"           Flask not running or wrong port
                               Make sure you ran: python app.py

Port 5000 already in use       Change port in app.py:
                               app.run(port=5001)

"API key not found" error      Create/edit .env file with:
                               OPENAI_API_KEY=sk-your-key

"Bot status = error"           Check .env file exists and has API key
                               Run: python test_bot.py

Styling not applied            Clear browser cache: Ctrl+Shift+Delete
                               Hard refresh: Ctrl+F5

Chat not responding            Check browser console (F12) for errors
                               Check bot status indicator

Can't connect to localhost     Check firewall settings
                               Try 127.0.0.1:5000 instead
