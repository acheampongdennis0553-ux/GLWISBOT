## 3-Minute Deployment Guide

### 1. Create Streamlit Account (1 minute)
- Go to: https://streamlit.io/cloud
- Click "Sign up"
- Select "Continue with GitHub"
- Authorize Streamlit to access your repos
- Confirm your email

### 2. Deploy Your App (1 minute)
- Click "New app"
- **Repository**: `acheampongdennis0553-ux/GLWISBOT`
- **Branch**: `master`
- **Main file path**: `streamlit_app.py`
- Click "Deploy"
- Wait 1-2 minutes for deployment ⏳

### 3. Add OpenAI API Key (1 minute)
- Go to Settings (gear icon) > Secrets
- Paste this in the text box:
```toml
OPENAI_API_KEY = "your-api-key-here"
```
- Replace with your actual OpenAI API key
- Save → App auto-restarts
- **Done! ✅**
